== README

Requirements:
Ruby 2.2.2p95
Rails 4.1.4

First time setup:
$ cd rails-blog/
$ bundle install
$ bundle exec rake db:create db:migrate ##it will create a mysql database and then migrate.
$ rails server -p 3001

##By default, the default server is WEBrick. YOu can change it with different rails server such as Thin server.
